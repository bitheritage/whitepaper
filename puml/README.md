## UML Diagrams Support

Markdown files can include UML diagrams. The diagrams are stored in the `puml` folder and 
written in [PlantUml](https://plantuml.com/).

To render the diagrams, the [markdown file](https://github.com/danielyaa5/puml-for-markdown) 
should include a link to the diagram file and just run:

```shell
puml-for-markdown
```
